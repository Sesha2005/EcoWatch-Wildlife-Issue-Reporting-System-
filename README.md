# 🌿 EcoWatch – Wildlife Issue Reporting System

EcoWatch is a full-stack web platform designed to bridge the gap between citizens and wildlife authorities. It enables users to report wildlife incidents in real time while allowing authorities to monitor, investigate, and resolve them efficiently.

> 🐾 Protecting wildlife through smarter reporting and faster response.

---

## 🚀 What EcoWatch Does

- 🧑 Citizen can report wildlife incidents (injured animals, sightings, threats)
- 🛠️ Authorities can investigate and update case status
- 📊 Dashboards provide clear tracking of all reports
- 🔄 Real-time status updates ensure transparency
- ⚡ SLA-based escalation improves response time

---

## 👥 User Roles

- **Citizen** → Submit reports & track status  
- **Authority** → Review and resolve cases  
- **Admin** → System overview and management  

---

## 🧩 Core Features

- 📝 Wildlife issue reporting with location and description  
- 📌 Status tracking: *Pending → In Progress → Resolved*  
- 📂 Organized dashboards for each role  
- 🔐 Secure authentication with encrypted passwords  
- ⚡ Escalation mechanism for delayed actions  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Java, Spring Boot (REST APIs)  
- **Database:** MySQL  

---

## 🔄 How It Works

1. Citizen submits a wildlife report  
2. Report is sent to backend via REST API  
3. Data is stored in MySQL database  
4. Authority reviews and updates status  
5. Citizen tracks updates in dashboard  

---

## 🔐 Security

- Passwords stored using **BCrypt encryption**  
- Role-based access control implemented
